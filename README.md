# CMPSC132
 Project Description:
 * A Single Player Number Guessing Game
 * Multiple Features:
    * Difficulty Levels
    * Dynamic Attempts Settings
    * Unlimited Gameplay
* Instructions:
    * Choose if you want to play the game (Y/N)
    * Choose the difficulty level
    * Depending on your difficultly level you will be given a specific number of attempts
        * Easy (10 Attempts)
        * Medium (5 Attempts)
        * Hard (3 Attempts)
    * Depending on your guess the computer will give you feedback ("Too High" or "Too Low")
    * After every guess the program will display the number of attempts you have remaining
    * If you exceed the number of attempts you lose
    * If you guess the correct number you win and it will display the number of attempts it took
    * Choose if you want to play another round (Y/N)
    * If N is selected you will exit out the game
